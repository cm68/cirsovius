Hilfsprogramme für (den RMAC) AssemblerTools for (the RMAC) Assembler

Während viele der hier vorgestellte Projekte in Assembler geschrieben sind — und somit relativ einfach zu verstehen —, sind viele Applikationen von Digital Research in einer Hochsprache1 geschrieben. Dies macht entsprechend Schwierigkeiten, den disassemblierten Code zu verstehen. Dies betrifft zwei Anwendungen, den Linker und das Bibliotheksprogramm. Ich habe diese Programme verwendet, weil beide den schnelleren Zugriff mit indizierten .REL-Dateien (.IRL-Dateien) auf Bibliotheken erlauben. LINK.COM2 erlaubt außerdem das Erstellen von RSXen.
Most of the presented utilities are written in pure assembler — therefore relatively easy to understand — but many of the Digital Research applications are written in a high level language1. However disassembled code of a high level language is much difficult to understand. In fact it concerns two utilities, the linker and the librarian. I used these programs because they allow faster library access using indexed .REL files (.IRL files). LINK.COM2 also allows the direct creation of RSX files
  • LINK :
    Hier habe ich „kosmetische Verbesserungen" durchgeführt. So gibt es jetzt eine Option [OX], die ohne Umwege .RSX-Dateien erzeugt. Dadurch ist die Option [OR] weggefallen. Ebenso fand ich die Option [B] überflüssig. Die neue Option [V] protokolliert den Ablauf auf dem Bildschirm. Die Option [S] durchsucht nun nicht nur eine Bibliothek im .REL- sondern auch im .IRL-Format.
    I did modify this utility slightly. Now the option [OX] does exist which creates an .RSX file the direct way. Therefore the option [OR] will not be used anymore. I also decided to disable the option [B]. The new option [V] logs activity on console. The option [S] now searches not only in an .REL library but also within an .IRL file.
  • LIB : Hier habe ich einige neue Optionen eingebaut:I did implement some new options:
    [E]:Zeigt neben den Public Symbolen auch die Externen Symbole eines Moduls an
    [U]:Zeigt in der Bibliothek nicht aufgelöste Referenzen an
    [R]:Zeigt Referenzen nach Sucheingabe von Labels in Modulen an
    [N]:Bildschirmausgabe erfolgt ohne Pause
    [E]:Displays Public symbols as well as External symbols within a module
    [U]:List unsolved references in a library
    [R]:List label references in modules by a given search mask
    [N]:Console output with No pause

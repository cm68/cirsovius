Diese Programme sind ausgelegt, um Binärdateien umzuwandeln in Quell texte (Typ .MAC beim DASM, .ASM beim REZ), die dann mit einem Z80 Assembler (DASM) oder 8080 Assembler (REZ) wieder zu lauffähigen Kode gewandelt werden können. Mit anderen Worten, ist eine .COM Datei vorhanden, die z.B. modifiziert werden soll, von der jedoch der Quelltext fehlt, dann sind dies genau die richtigen Programme. Während bei Arbeiten mit REZ die CP/M PLUS Assembler MAC oder RMAC genügen, benötigt man zum Arbeiten mit DASM einen Z80 Assembler, also z.B. den M80 von MICROSOFT oder ZMAC aus dem PD Tool. These programs are designed to convert binary files into source texts (.MAC type for DASM, .ASM for REZ), which can then be reassembled into executable code using a Z80 assembler (DASM) or an 8080 assembler (REZ). In other words, if you have a .COM file that needs to be modified but the source code is missing, then these are exactly the right programs. While working with REZ the CP/M PLUS assemblers MAC or RMAC are sufficient, working with DASM requires a Z80 assembler, e.g. M80 from MICROSOFT or ZMAC from PD Tools.

Da die Regeln zum Arbeiten mit beiden Programmen gleich sind, wird hier als Beispiel DASM erläutert. Bei Abweichungen wird dies gesondert angedeutet. Allgemein gilt, daß REZ älter und weniger komfortabel ist als DASM. Since the rules for working with both programs are the same, DASM is used as an example here. Any differences will be noted separately. In general, REZ is older and less convenient than DASM.
(Im Anhang sind einige Änderungen angegeben, die sich speziell auf eine verbesserten Version von DASM beziehen.The appendix lists some changes that specifically relate to an improved version of DASM.)

Wie ist nun die Arbeitsweise von DASM? How does DASM work?
Aus der zu bearbeitenden ,COM Datei werden zunächst drei weitere Dateien erzeugt: From the .COM file to be processed, three additional files are first generated:

-.SYM :Diese Datei besteht aus Informationen über LabelsThis file contains information about labels
-.CTL :Diese Datei besteht aus Informationen über Programmcode oder KonstantenThis file contains information about program code or constants
-.DOC :Diese Datei enthält optionalen KommentarThis file contains optional comments

Der Vorteil dieser Zerlegung in einzelne Dateien ist, daß diese sehr kurz sind. Erst am Ende einer Bearbeitung, die durchaus über mehrere Tage gehen kann, wird die endgültige Quelldatei .MAC erzeugt. Diese kann, je nach Umfang der .COM Datei sehr lang sein. Realistisch ist ein Faktor 4 bis 8, d.h. eine .COM Datei mit dem Umfang 8kByte erzeugt dann eine .MAC Datei mit 32-64kByte. The advantage of this decomposition into individual files is that they are very short. Only at the end of the editing process, which may well span several days, is the final source file .MAC generated. This can be very long depending on the size of the .COM file. A realistic factor is 4 to 8, i.e. a .COM file of 8kByte will produce a .MAC file of 32-64kByte.

B sucht, im angegebenen Bereich, wie bei A, nach Labels. D.h. bei Sprungbefehlen und Unterprogrammaufrufen wird anstelle der Adresse ein Label eingesetzt. Das gilt natürlich dann auch für die Adresse selber. Das Label hat die Form Lxxxx, wobei xxxx die Adresse in hexadezimaler Form darstellt. B searches the specified range, like A, for labels. That is, for jump instructions and subroutine calls, a label is inserted instead of the address. This naturally also applies to the address itself. The label has the form Lxxxx, where xxxx represents the address in hexadecimal form.

Neue DASM VersionNew DASM version

Es existiert eine neuere Version von DASM, die sich in einigen Punkten von der beschriebenen Version unterscheidet: A newer version of DASM exists that differs in some points from the described version:
  1. Die Labeldatei erhält anstelle der Extension .SYM die neue Extension .SMB. Dadurch wird das übeschreiben der Labeldatei durch den CP/M Linker LINK verhindert, der als Labeldatei ebenfalls die Extension .SYM verwendet.The label file receives the new extension .SMB instead of .SYM.

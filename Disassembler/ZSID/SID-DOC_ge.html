Generierung einer (Z)SID-Version für 8080 oder Z80Generating a (Z)SID version for 8080 or Z80

 Folgende Schritte müssen für die Erstellung einer neue Version des 8080- oder Z80-SID durchgeführt werden: The following steps must be performed to create a new version of the 8080 or Z80 SID:

Die Datei XSID.LIB muss editiert werden für die korrekte Einstellung der Version.
Siehe in dieser Datei die "EQU" für das Setzen auf TRUE oder FALSE. The file XSID.LIB must be edited for the correct version settings.
See the "EQU" directives in this file for setting to TRUE or FALSE.

Editieren der Loaderdatei XSIDLOAD.MAC falls notwendig, assemblieren und linken mit Edit the loader file XSIDLOAD.MAC if necessary, then assemble and link with

M80 =XSIDLOAD
LINK XSIDLOAD
Das (Z)SID-Paket besteht aus vier Dateien, die gegebenenfalls editiert werden müssen: The (Z)SID package consists of four files that may need to be edited:

SIDRELO.MACDer KernThe core
SIDLA.MACDas 8080 Assemble/List-PaketThe 8080 assemble/list package
ZSIDLA.MACDas Z80 Assemble/List-PaketThe Z80 assemble/list package
SIDCMD.MACDer KommandozeileninterpreterThe command line interpreter

Das Paket wird assembliert und gelinkt mit The package is assembled and linked with

M80 =SIDRELO<<-- DIE 4 ODER 8 FOLGENDEN FEHLER IGNORIEREN:IGNORE THE FOLLOWING 4 OR 8 ERRORS:
R xxxx'                  C+     PC	aset	$ and 00ffh
R                        C+     IF	PC GT 07fh
R                        C+     IF	PC GT 0
R xxxx'                  C+     	ds	.ds-PC,0
Diese werden ein- oder zweimal ausgegeben. These are output once or twice.

LINK SIDRELO[OS]<<-- WICHTIGE OPTIONIMPORTANT OPTION

!!!ACHTUNG:WARNING:Die ZCPR-Version von SID benötigt für die BIOS-Tabellen Seitengrenzen (xx00H)The ZCPR version of SID requires page boundaries (xx00H) for the BIOS tables
!!!Die Symbole BIOS und .BIOS beachten, die dicht beieinander liegen müssen.Note the symbols BIOS and .BIOS, which must be close together.

Fertigstellen des Ladeprogramms und des Debuggers mit Finalize the loader and the debugger with

LDBLD XSIDLOAD SIDRELO NEUE_DATEI

NEUE_DATEI ist dann der fertige ausführbare Debugger. NEUE_DATEI is then the finished executable debugger.

DisassemblerübersichtDisassembler Overview

Die folgenden Disassembler sind mir noch bekannt:The following disassembler tools are additionally known to me:

ProgrammProgramProgrammiererProgrammerVersionDatumDateBeschreibungDescription
1. Z80DIS Kenneth Gielow 2.2 18.4.1987 Dieser Disassembler führt seine Arbeit sehr präzise aus und basiert auf der Untersuchung des zu bearbeitenden Programmes (Künstliche Intelligenz). Neben dem automatischen Disassemblieren ist auch ein manueller Eingriff möglich.This disassembler works very precise and bases upon the analysis of the inspected program (artificial intelligence). Operator may intervent manually besides the automatic disassembling.
2.DZ John Washington   29.11.1986 Dieses Programm (auch bekannt als DazzleStar) kommt mit einer Oberfläche wie WordStar.This program (also known as DazzleStar) uses a WordStar alike user interface.
3. PRODIZ80 Rolf Keller 1.0 1987 Ähnlich wie Z80DIS erzeugt dieses Programm den Quell-Code automatisch.The program creates source code automatically such as Z80DIS.
Erst viel später lernte ich noch weitere Disassembler kennen:At a later time I found some more disassemblers:
4. DIS MICRO COM, INC.   1978 Ähnlich wie Z80DIS erzeugt dieses Programm den Quell-Code automatisch.The program creates source code automatically such as Z80DIS.
5. REZILOG   0.9 1986 Ähnlich DASM.Similar to DASM.
6. REASS W. Dames   1985 Ähnlich wie Z80DIS erzeugt dieses Programm den Quell-Code automatisch. Wie bei (Z)SID wird der eigentliche Disassembler in den oberen Speicher verschoben. Dieses Programm ist eine Entwicklung aus der ehemaligen DDR.The program creates source code automatically such as Z80DIS. As known by (Z)SID the disassembler will be moved to top of memory. This tool is a design of the former GDR.
7. REVAS A. E. Hawley 2.5 1978 Ähnlich wie Z80DIS erzeugt dieses Programm den Quell-Code automatisch. Wie bei (Z)SID wird ein Lader für den Disassembler in den oberen Speicher verschoben. Der eigentliche Disassembler wird hinter das zu bearbeitende Programm geladen.The program creates source code automatically such as Z80DIS. As known by (Z)SID a loader will be moved to top of memory. The actual disassembler will be installed after the program to be disassembled.

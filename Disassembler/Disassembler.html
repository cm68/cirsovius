Als Grundlage für den Disassembler habe ich DASM aus dem Public Domain-Bereich gewählt. Das — aus meiner Sicht — Gute an diesem Programm ist, dass man sich bei der Arbeit mit dem zu disassemblierenden Programm auseinandersetzen muss.
Leider war das Programm in seiner Funktionalität eingeschränkt, so dass ich den Code erweitert habe. Als Projekte für die Disassemblierung sind hier einige weitere Programme abgedruckt.
Wegen der Speicherbeschränkung und dem möglichst komfortablen Umgang mit dem Editor hatte ich alle Dateien in Portionen von ca. 16k aufgeteilt — zusammengefügt wurden diese dann mit INCLUDE-Befehlen.
As base for disassembling I selected the program DASM out of the Public Domain. From my point of view this program is very good because one has to deal with the code during disassembly.
Unfortunately the functionality was restricted — generally the memory management — so I expanded the code. Find listed below some of my disassembly projects.
Because the length of the source code exceeded the available memory for editing it, I splitted the files into portions of about 16K — the main files use INCLUDE-instructions to put them together again.

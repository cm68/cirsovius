LENG.MAC
@LENG B: Bestimmen der Länge einer ZeichenketteCompute length of string
  I: Das Registerpaar HL hält die Adresse auf die Zeichenkette.Reg pair HL points to string closed by zero.
  O: Das Register B und der Akku enthalten die Länge der Kette, die Zero Flag ist gesetzt, falls die Länge Null ist.Reg B and Accu hold length of string, Zero flag is set if the length is zero.
ENDSTR.MAC
@ENDST B: Zeichenkette auf Ende setzen.Position to end of string.
  I: Registerpaar DE zeigt auf Zeichenkette.Reg pair DE points to string closed by zero.
  O: Register DE zeigt auf das Ende der Zeichenkette.Reg DE points to the closing zero of the string.

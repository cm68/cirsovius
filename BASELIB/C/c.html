C-RoutinenC Routines

Die hier vorgestellten Routinen sind solche für die Verabeitung von Routinen, die an die Sprache C angelehnt sind. Die meisten Routinen benötigen die Library SYSEQU.LIB, die mittels MACLIB SYSEQU.LIB und einige die Library C.LIB, die mittels MACLIB C.LIB eingefügt werden. Nicht alle Routinen sind von mir entwickelt sondern lediglich in meine Bibliothek übernommen worden: Einige Hinweise zu den Routinen finden sich hier.The routines presented here are those for processing routines modeled after the C language. Most routines require the library SYSEQU.LIB, which is included via MACLIB SYSEQU.LIB, and some require the library C.LIB, which is included via MACLIB C.LIB. Not all routines were developed by me; some were merely adopted into my library: Some notes on the routines can be found here.

PRINTF.MAC
PRINTF B: Formatierung einer Zeichenkette im C StilFormat a string in C style
  I: Registerpaar HL zeigt auf zu formatierende Zeichenkette, abgeschlossen mit Null. Registerpaar DE zeigt auf Kontroll-Adress-Feld. Registerpaar BC zeigt auf die Ergebniszeichenkette, das erste Byte hält die maximale Länge der Kette.
Die zu formatierende Zeichenkette kann folgende Kontrollzeichen beinhalten:
\tTabulator setzen
\bBackspace setzen
\rCarriage Return setzen
\fForm Feed setzen
\nNeue Zeile setzen
\\Zeichen „\" übernehmen
 Alle anderen Kombination werden übernommen
Mit Variabler aus Kontroll-Adress-Feld:
%cAls Zeichen einfügen
%sAls Zeichenkette einfügen
%dAls Ganzzahl mit Vorzeichen einfügen
%uAls vorzeichenlose Ganzzahl einfügen
%xAls Hexzahl einfügen
%%Zeichen „%" übernehmen
 Alle anderen Kombination werden übernommen
Register pair HL points to the string to be formatted, terminated with null. Register pair DE points to the control address array. Register pair BC points to the result string; the first byte holds the maximum length of the string.
The string to be formatted can contain the following control sequences:
\tInsert tab
\bInsert backspace
\rInsert carriage return
\fInsert form feed
\nInsert new line
\\Insert literal „\" character
 All other combinations are passed through unchanged
With variable from the control address array:
%cInsert as character
%sInsert as string
%dInsert as signed integer
%uInsert as unsigned integer
%xInsert as hexadecimal number
%%Insert literal „%" character
 All other combinations are passed through unchanged
  O: Die Ergebniskette ist gefüllt, Abschluss mit Null. Die gesetzte Carryflag zeigt Überlauf an.The result string is filled, terminated with null. The carry flag set indicates overflow.

SBRK.MAC
SBRK B: Bereitstellen von neuem SpeicherplatzAllocate new memory
  I: Registerpaar HL hält Anzahl gewünschter Bytes.Register pair HL holds the number of requested bytes.
  O: Registerpaar HL hält Adresse auf freien Speicher. Falls nicht genügend Speicher vorhanden ist, hält Registerpaar HL 0.
Der Stack Pointer muss mit der höchsten möglichen Adresse geladen sein (aus Adresse 0006). In SBRK-2 steht der zu berücksichtigende Platz für den Stack (Default 1024). Der freie Speicherplatz wird ermittelt aus der Assembler Variablen $MEMRY.Register pair HL holds the address of the free memory. If not enough memory is available, register pair HL holds 0.
The stack pointer must be loaded with the highest possible address (from address 0006). SBRK-2 contains the space to reserve for the stack (default 1024). The free memory is determined from the assembler variable $MEMRY.

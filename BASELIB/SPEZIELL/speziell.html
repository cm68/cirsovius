Bildschirm-RoutinenScreen Routines

Die hier vorgestellten Routinen sind solche für das Arbeiten mit Bildschirm-Routinen.The routines presented here are for working with screen routines.
Modul-EntryModule Entry B: BeschreibungDescription, I: Eingabe-Input, O: AusgabeparameterOutput parameters
FLASH.MAC
FLASH B: Anzeige zweier Texte, wobei der erste Text invers blinkt.Flash text on screen until a user key is detected. Displays two texts, where the first text blinks in inverse video.
  I: Register L hält die X Position und Register H die Y Position des Start Textes. Registerpaar DE zeigt auf zwei hintereinander liegende Texte, die jeweils durch eine Null beendet werden. Registerpaar BC zeigt auf vier sequentielle und durch Nullen getrennte SteuersequenzenRegister L holds the X position and register H holds the Y position of the cursor. Register pair DE points to two consecutive texts, each terminated by a zero. Register pair BC points to four sequential, zero-terminated control sequences: 1 - Make cursor invisible, 2 - Make cursor visible, 3 - Turn inverse on, 4 - Turn inverse off.
  O: ---
CFLASH.MAC
@FLASH B: Anzeige zweier Texte, wobei der erste Text invers blinkt, mit voreingestellter Kontrollsequenz.Flash text on screen until a user key is detected, with a default control sequence. Displays two texts, where the first text blinks in inverse video, using a preset control sequence.
  I: Der Aufruf erfolgt wie beim Modul FLASH jedoch wird das Registerpaar BC intern auf eine JOYCE kompatible Sequenz voreingestellt.The call is made the same way as the FLASH module, but register pair BC is internally preset to a JOYCE-compatible sequence.
  O: ---

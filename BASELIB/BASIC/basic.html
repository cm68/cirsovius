BASIC-RoutinenBASIC Routines

Die hier vorgestellten Routinen sind solche für die Verabeitung von Routinen, die an die Sprache BASIC angelehnt sind:The routines presented here are those for processing functions modeled on the BASIC language:
Modul-EntryModule Entry B: BeschreibungDescription, I: Eingabe-Input, O: AusgabeparameterOutput parameters
MID.MAC
@MID B: Teil einer Zeichenkette ladenGet MID$ of a string
  I: Registerpaar HL zeigt auf Originalzeichenkette, Registerpaar DE zeigt auf neue Kette, Register C weist auf die Startposition und Register B hält die Anzahl zu ladender Zeichen. Ist Register B Null, so werden alle Zeichen bis zum Ende geladen.Register pair HL points to the original string, register pair DE points to the new string, register C points to the start position and register B holds the number of characters to load. If register B is zero, all characters to the end will be copied.
  O: Bei gesetzter Carry Flag liegt ein Fehler vor, das Register C weist auf eine Position, die größer ist als die Länge der Zeichenkette.If the carry flag is set, an error has occurred: register C points to a position that is greater than the length of the string.

INSTR.MAC
@INSTR B: Suchen einer TeilzeichenketteFind substring from target string
  I: Registerpaar HL zeigt auf Zeichenkette, in der gesucht werden soll, Registerpaar DE zeigt auf zu suchende Kette. Register C hält Startposition in der Suchkette. Ist C Null, so wird mit dem ersten Zeichen begonnen.Register pair HL points to the target string to be searched, register pair DE points to the substring to search for. Register C holds the start position within the target string. If C is zero, the search begins at the first character.
  O: Register B hält die Position der zu suchenden Kette und die Zero Flag ist gesetzt. Bei nicht gesetzter Zero Flag, ist die zu suchende Kette nicht gefunden worden.Register B holds the position of the found substring and the zero flag is set. If the zero flag is not set, the substring was not found.

  O: Zeichenkette mit konstanten Zeichen gefüllt. Das Registerpaar DE zeigt auf die abschließende NullString filled with the constant character. Register pair DE points to the terminating zero

OPENI.MAC
@OPENI B: Öffnen einer vorhandenen Datei zum LesenOpen an existing file for reading
  I: Das Registerpaar HL zeigt auf eine Zeichenkette, die den Dateinamen, die Extension und eventuell das Laufwerk beinhaltet. Das Registerpaar DE zeigt auf einen File Control Block (FCB), der 36 Bytes lang sein muss. Register C hält die Kanalnummer im Bereich 1 bis 15. Falls Register C 0 ist, wird kein Kanal geöffnet sondern nur der FCB.Register pair HL points to a string containing the file name, extension and optionally the drive. Register pair DE points to a File Control Block (FCB), which must be 36 bytes long. Register C holds the channel number in the range 1 to 15. If register C is 0, no channel is opened and only the FCB is set up.
  O: Die gesetzte Carry Flag zeigt einen Fehler an. Hierbei hält der Akku folgende Werte:
52Falsche Kanalnummer (>15)
53Datei ist nicht vorhanden
55Der Kanal ist bereits geöffnet
65Der Dateiname ist fehlerhaft
A set carry flag indicates an error. The accumulator holds the following values:
52Invalid channel number (>15)
53File not found
55Channel is already open
65File name is invalid

FIND.MAC
@FIND B: Suchen einer DateiSearch for a file
  I: Das Registerpaar HL zeigt auf eine Zeichenkette, die den Dateinamen, die Extension und eventuell das Laufwerk beinhaltet. Das Registerpaar DE zeigt auf die genau 12 Byte lange resultierende Zeichenkette. Register B hält eine optionale Zahl, die die Anzahl von Versuchen zum Suchen angibt. Dies ist sinnvoll bei Universalzeichen im Dateinamen.Register pair HL points to a string containing the file name, extension and optionally the drive. Register pair DE points to the resulting string, which is exactly 12 bytes long. Register B holds an optional number indicating how many search attempts to perform. This is useful when wildcard characters are used in the file name.
  O: Die gesetzte Carry Flag zeigt einen Fehler an oder dass die gesuchte Datei nicht gefunden wurde. Hierbei hält der Akku folgende Werte:
0Datei wurde nicht gefunden
64Der Dateiname ist fehlerhaft
Wurde die Datei nicht gefunden, so ist die resultierende Zeichenkette leer. Andernfalls ist diese Kette exakt mit 12 Bytes gefüllt, das Format ist NNNNNNNN.XXXA set carry flag indicates an error or that the file was not found. The accumulator holds the following values:
0File was not found
64File name is invalid
If the file was not found, the resulting string is empty. Otherwise, the string is filled with exactly 12 bytes in the format NNNNNNNN.XXX

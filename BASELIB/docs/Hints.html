Hinweise zur CPU-Bestimmung

Im allgemeinen ist die Bibliothek in Z80 Code geschrieben. Dies betrifft in der Hauptsache relative Sprünge. Am Anfang eines Programmes, das die Bibliothek nutzt, sollte deshalb folgende Sequenz stehen: 


        SUB     A       ; Überprüfen Z80
        JP      PO,OK   ; Sprung wenn richtige CPU

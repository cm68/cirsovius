Hinweise zur CPU-BestimmungNotes on CPU Detection

Im allgemeinen ist die Bibliothek in Z80 Code geschrieben. Dies betrifft in der Hauptsache relative Sprünge. Am Anfang eines Programmes, das die Bibliothek nutzt, sollte deshalb folgende Sequenz stehen:In general, the library is written in Z80 code. This mainly concerns relative jumps. At the beginning of a program that uses the library, the following sequence should therefore be placed: 


        SUB     A       ; Überprüfen Z80Check for Z80
        JP      PO,OK   ; Sprung wenn richtige CPUJump if correct CPU

Hinweise zu den Disk-Routinen mit Random Record DateienNotes on the Disk Routines with Random Record Files

Die Random Rekord Dateien werden über einen Parameterblock angesprochen. Dieser wird in der Routine $RNDRG in die CPU Register geladen. Aufbau und Register sind:The random record files are accessed through a parameter block. This is loaded into the CPU registers in the routine $RNDRG. The structure and registers are:

Registerpaar DE hält Parameter Block und wird in das Register IX geladen.Register pair DE holds the parameter block and is loaded into register IX.

Bei der Initialisierung einer Datei mit INIREC und FIXREC steht in PB+1,2 die maximale Anzahl Rekords. Es werden bei der Initialisierung neue Rekords beschrieben mit dem Inhalt des Rekord Daten Puffers.During the initialization of a file with INIREC and FIXREC, PB+1,2 contains the maximum number of records. During initialization, new records are written with the contents of the record data buffer.
FIXREC benötigt den höchsten Rekord in PB+9,10. Abhängig vom höchsten Rekord und dem gewünschten wird die Datei entweder verkürzt oder erweitert.FIXREC requires the highest record in PB+9,10. Depending on the highest record and the desired one, the file is either truncated or extended.

Working with High-Level Language RoutinesDas Arbeiten mit Hochsprach-Routinen

This collection was created after I had programmed in several high-level languages. Afterwards I was missing routines from the languages BASIC, C, and PASCAL, which I then implemented.Diese Sammlung entstand, nachdem ich in einigen Hochsprachen programmiert hatte. Danch „fehlten" mir Routinen aus den Sprachen BASIC, C und PASCAL, die ich dann implemetierte.
  • BASIC : Routines modeled on the BASIC languageRoutinen, die an die Sprache BASIC angelehnt sind
  • Pascal : Routines modeled on the PASCAL languageRoutinen, die an die Sprache PASCAL angelehnt sind
  • BAS-Pas : Routines modeled on the BASIC and Pascal languagesRoutinen, die an die Sprache BASIC und Pascal angelehnt sind
  • C : Routines modeled on the C languageRoutinen, die an die Sprache C angelehnt sind

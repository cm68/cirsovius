Das Arbeiten mit Hochsprach-Routinen

Diese Sammlung entstand, nachdem ich in einigen Hochsprachen programmiert hatte. Danch „fehlten" mir Routinen aus den Sprachen BASIC, C und PASCAL, die ich dann implemetierte.
  • BASIC : Routinen, die an die Sprache BASIC angelehnt sind
  • Pascal : Routinen, die an die Sprache PASCAL angelehnt sind
  • BAS-Pas : Routinen, die an die Sprache BASIC und Pascal angelehnt sind
  • C : Routinen, die an die Sprache C angelehnt sind

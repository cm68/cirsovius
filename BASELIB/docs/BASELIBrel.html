Aufbau der LibraryStructure of the Library

Bei der Gestaltung der Bibliothek ist es wichtig, auf eine bestimmte Reihenfolge zu achten. So erhielten alle Module, die keinen externen Aufruf beinhalteten, den Level 1, Module, die einen externen Aufruf auf Module des Levels 1 beinhalteten, erhielten den Level 2, Module, die einen externen Aufruf auf Module des Levels 1 oder 2 beinhalteten, erhielten den Level 3 usw. Module mit Level 1 wurden in eine Bibliothek BASE1.REL gepackt, Module mit Level 2 wurden in eine Bibliothek BASE2.REL, usw. Zum Schluss wurden alle Module in eine große Bibliothek gepackt, undzwar in umgekehrter Reihenfolge. Bei LIB sah das Kommando dafür z.B. so aus:When designing the library, it is important to maintain a specific ordering. All modules that contained no external calls were assigned Level 1, modules that contained an external call to Level 1 modules were assigned Level 2, modules that contained an external call to Level 1 or 2 modules were assigned Level 3, etc. Level 1 modules were packed into a library BASE1.REL, Level 2 modules into a library BASE2.REL, etc. Finally, all modules were packed into one large library, in reverse order. For LIB, the command looked e.g. like this:
LIB BASE=BASE3,BASE2,BASE1

In den einzelnen Modulen findet sich am Anfang z.B. folgender Dreizeiler:At the beginning of each module, you will find e.g. the following three-liner: 
	title	Module from library BASE_1 : CONIN
	name	('CONIN')
	maclib	baselib.lib
Hierbei gibt BASE_1 an, dass es sich um ein Modul aus dem Level 1 der Gruppe BASE handelt. In der Library heißt dieses Modul CONIN.Here, BASE_1 indicates that this is a module from Level 1 of the BASE group. In the library, this module is named CONIN.

Als grundlegende Gruppe diente die schon erwähnte Gruppe BASE, die aus 7 Levels besteht. Es ergeben sich folgende Abhängigkeiten:The fundamental group is the already mentioned BASE group, which consists of 7 levels. The following dependencies result:

Danach kann dann die Bibliothek zusammengestellt werden mit z.B. dem KommandoThis means that the modules of the BASE group must be added to the library last. As stated above, all groups must be created first. After that, the library can be assembled with e.g. the command
LIB BASELIB=PDM,C,INT,HIFLOAT,CPX,HILIB,BASE[I]

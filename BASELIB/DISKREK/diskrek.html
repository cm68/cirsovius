Routinen für rekordweise DiskettenoperationenRoutines for record-based disk operations

Die hier vorgestellten Routinen sind solche für zur Rekord Ein-/Ausgabe bei Diskettenoperationen.The routines presented here are those for record input/output during disk operations.
Modul-EntryModule Entry B: BeschreibungDescription, I: Eingabe-Input, O: AusgabeparameterOutput parameters
DSKRED.MAC
DSKRED B: Lesen eines sequentiellen Rekords von DisketteRead a sequential record from disk
  I: Registerpaar DE hält Adresse des FCBRegister pair DE holds the address of the FCB
  O: Carryflag gesetzt falls Datei nicht gefunden oder vollständig gelesen wurde, sonst nicht gesetzt.Carry flag is set if the file was not found or has been completely read, otherwise not set.
DSKWRT.MAC
DSKWRT B: Schreiben eines sequentiellen Rekords auf DisketteWrite a sequential record to disk
  I: Registerpaar DE hält Adresse des FCBRegister pair DE holds the address of the FCB
  O: Carryflag ist gesetzt, falls Datei nicht gefunden wurde oder kein Platz mehr auf der Diskette ist, sonst nicht gesetzt.Carry flag is set if the file was not found or there is no more space on the disk, otherwise not set.
RNDRED.MAC
RNDRED B: Lesen eines wahlweisen Rekords von DisketteRead a random record from disk
  I: Registerpaar DE hält Adresse des FCBRegister pair DE holds the address of the FCB
  O: Carryflag gesetzt, falls keine weiteren Rekords existieren oder die Datei nicht auf Diskette ist, sonst nicht gesetzt.Carry flag is set if no more records exist or the file is not on disk, otherwise not set.
RNDWRT.MAC
RNDWRT B: Schreiben eines wahlweisen Rekords auf DisketteWrite a random record to disk
  I: Registerpaar DE hält Adresse des FCBRegister pair DE holds the address of the FCB
  O: Carryflag ist gesetzt, falls Datei nicht gefunden wurde oder kein Platz mehr auf der Diskette ist, sonst nicht gesetzt.Carry flag is set if the file was not found or there is no more space on the disk, otherwise not set.
FILEIO.MAC
@FILIO B: Ausführen spezieller DiskettenfunktionenExecute special disk functions
  I: Registerpaar DE hält Adresse des FCB und Akku hält die auszuführende BDOS FunktionRegister pair DE holds the address of the FCB and the accumulator holds the BDOS function to execute
  O: Carryflag ist gesetzt falls die BDOS Funktion fehlerhaft ist (Code nicht Null), sonst nicht gesetzt. Akku hält die Nummer des Extended ErrorkodeCarry flag is set if the BDOS function returned an error (code not zero), otherwise not set. The accumulator holds the extended error code number
SEARCH.MAC
SRCFRS SRFBUF B: Suche nach erstem EintragSearch for first entry
SRCNXT   B: Suche nach nächstem EintragSearch for next entry
  I: Registerpaar DE hält Adresse des FCB bei erstem Aufruf, sonst beliebig. Das Feld SRFBUF muss die Adresse eines 128 Byte Puffers halten, die Voreinstellung ist der Standard DMA Puffer ab 0080HRegister pair DE holds the address of the FCB on the first call, otherwise any value. The field SRFBUF must hold the address of a 128-byte buffer; the default is the standard DMA buffer at 0080H
  O: Im Normalfall ist die Carryflag nicht gesetzt und Registerpaar HL hält die Adresse eines Namensfeldes einer FCB. Die Carryflag ist gesetzt, falls kein weiterer Eintrag gefunden wurde, oder bei einem Fehler.
Zwischen den Aufrufen darf keine andere Diskettenfunktion ausgeführt werdenNormally the carry flag is not set and register pair HL holds the address of a name field of an FCB. The carry flag is set if no further entry was found, or on an error.
No other disk function may be executed between calls
SRCFIL.MAC
SRCFIL B: Suche nach Einträgen mit der Erlaubnis der Ausführung einer DiskettenfunktionSearch for entries with permission to execute a disk function in between
  I: Registerpaar DE hält Adresse des FCB bei erstem Aufruf, sonst beliebig. Vor dem ersten Aufruf muss das Byte in SRCFIL-1 auf Null gesetzt sein.Register pair DE holds the address of the FCB on the first call, otherwise any value. Before the first call, the byte at SRCFIL-1 must be set to zero.
  O: Im Normalfall ist die Carryflag nicht gesetzt und der gewählte FCB ist gefüllt mit der gefundenen Datei. Die Carryflag ist gesetzt, falls kein weiterer Eintrag gefunden wurde, oder bei einem Fehler.Normally the carry flag is not set and the specified FCB is filled with the found file. The carry flag is set if no further entry was found, or on an error.
RMLSEC.MAC
RMLSEC B: Lesen mehrerer sequentieller Rekords von DisketteRead multiple sequential records from disk
  I: Registerpaar DE hält Adresse des FCBRegister pair DE holds the address of the FCB
  O: Carryflag gesetzt falls Datei nicht gefunden oder ein Lesefehler gefunden wurde, sonst nicht gesetzt. Im Extended Mode Verzweigung auf Fehlerroutine möglich. Wenn die Carryflag nicht gesetzt ist, so zeigt die Zeroflag das Leseergebnis an:
Flag gesetzt: Alle gewählten Sektoren sind gelesen worden
Flag nicht gesetzt: Es wurde das Dateiende gefunden, der Akku hält die Anzahl gelesener Rekords, wobei auch Null als Ergebnis vorkommen kannCarry flag is set if the file was not found or a read error occurred, otherwise not set. In extended mode, a branch to the error routine is possible. If the carry flag is not set, the zero flag indicates the read result:
Flag set: All selected sectors have been read
Flag not set: End of file was found; the accumulator holds the number of records read, which may also be zero

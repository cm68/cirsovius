Routinen für die ZahlenwandlungRoutines for Number Conversion

Die hier vorgestellten Routinen sind solche für Zahlenwandlungen.The routines presented here are those for number conversions.
Modul-EntryModule Entry B: BeschreibungDescription, I: Eingabe-Input, O: AusgabeparameterOutput parameters
HEXNIB.MAC
HEXNIB B: Wandeln eines Hexwertes aus einer ZeichenketteGet hex nibble from an ASCII character string
  I: Registerpaar HL hält Adresse der ZeichenketteRegister pair HL points to the ASCII buffer
  O: Akku hält den Hexwert, Registerpaar HL zeigt auf nächstes ZeichenAccumulator holds the hex nibble, register pair HL points to next character
HEXBYT.MAC
HEXBYT B: Wandeln eines Hexbytes aus einer ZeichenketteGet hex byte from an ASCII stream
  I: Registerpaar HL hält Adresse der Zeichenkette. Registerpaar DE hält Adresse eines HexpuffersRegister pair HL points to the ASCII buffer. Register pair DE points to the hex buffer
  O: Der Hexpuffer ist gefüllt mit dem Byte, beide Adressen entsprechend inkrementiertThe hex buffer is filled with the byte, both addresses incremented accordingly
HEXIN.MAC
HEXIN B: Wandeln eines Hexwertes aus einer ZeichenketteInput a 16 bit hex word from an ASCII character string
  I: Registerpaar DE hält Adresse der Zeichenkette. Register B hält die Anzeige des Endes der KetteRegister pair DE points to the ASCII buffer. Register B holds the end of string marker
  O: Registerpaar HL hält einen entsprechenden Hexwert. Bei Überlauf des Wertes oder falschem Zeichen in der Kette wird die Carry Flag gesetztRegister pair HL holds the 16 bit result. Carry flag is set on overflow or illegal character
ASCNIB.MAC
ASCNIB B: Wandeln eines Hexwertes in ZeichenGet ASCII character from hex nibble
  I: Der niederwertige Teil des Akkus hält Hexwert. Registerpaar DE hält Adresse des ZeichenpuffersLower part of accumulator holds hex nibble. Register pair DE points to the ASCII buffer
  O: Das Zeichen ist im Puffer gespeichert, Puffer inkrementiertThe ASCII character is stored into the buffer, buffer updated
ASCBYT.MAC
ASCBYT B: Wandeln eines Hexbytes in ZeichenGet ASCII characters from hex byte
  I: Registerpaar DE hält Adresse des Zeichenpuffers. Registerpaar HL hält Adresse des HexpuffersRegister pair DE points to the ASCII buffer. Register pair HL points to the hex buffer
  O: Die Zeichen sind im Puffer gespeichert, Puffer sind inkrementiertThe ASCII buffer is filled with two characters, buffers updated
HEXOUT.MAC
HEXOUT B: Wandeln eines Hexwertes in eine ZeichenketteConvert a 16 bit number to hex character string
  I: Registerpaar DE hält Adresse der Zeichenkette. Registerpaar HL hält 16 Bit WertRegister pair DE points to the ASCII buffer. Register pair HL holds the 16 bit number
  O: Die Zeichenkette ist gefüllt mit vier Bytes, die Adresse entsprechend inkrementiertThe ASCII buffer is filled with four characters, buffer updated
DECOUT.MAC
DECOUT B: Wandeln einer vorzeichenlosen 16 Bit Ganzzahl in ZeichenConvert an unsigned 16 bit integer to ASCII decimal string
  I: Registerpaar HL hält die 16 Bit Zahl. Registerpaar DE hält Adresse des Zeichenpuffers. Register B hält Zeichen, das am Ende gespeichert wird.Register pair HL holds the 16 bit number. Register pair DE points to the ASCII buffer. Register B holds the character to be stored at the end (usually $ or zero).
  O: Puffer mit numerischen Zeichen (0 .. 9) gefüllt und abgeschlossen mit Zeichen in Register B. Das Register HL zeigt auf das Abschlusszeichen.Buffer filled with numeric characters (0 .. 9) and closed by the character in register B. Register HL points to the close location.
SGNOUT.MAC
SGNOUT B: Wandeln einer 16 Bit Ganzzahl in ZeichenConvert a signed 16 bit integer to ASCII decimal string
  I: Registerpaar HL hält die 16 Bit Zahl. Registerpaar DE hält Adresse des Zeichenpuffers. Register B hält Zeichen, das am Ende gespeichert wird.Register pair HL holds the 16 bit number. Register pair DE points to the ASCII buffer. Register B holds the character to be stored at the end (usually $ or zero).
  O: Puffer mit numerischen Zeichen (0 .. 9) gefüllt und abgeschlossen mit Zeichen in Register B. Das Register HL zeigt auf das Abschlusszeichen.Buffer filled with numeric characters (0 .. 9) and closed by the character in register B. Register HL points to the close location.
DECIN.MAC
DECIN B: Wandeln einer Zeichenkette in vorzeichenlose 16 Bit GanzzahlConvert an ASCII decimal string to unsigned 16 bit integer
  I: Registerpaar DE hält Adresse der Zeichenkette. Register B hält Endzeichen der Zeichenkette, bei -1 endet Wandlung bei erstem nichtnumerischen ZeichenRegister pair DE points to the ASCII buffer. Register B holds the end marker of the string; if -1, conversion ends at the first non-numeric character
  O: Registerpaar DE zeigt auf Abbruchzeichen, Registerpaar HL hält 16 Bit Zahl bei nicht gesetzter Carryflag. Carryflag ist gesetzt bei Überlauf oder falschem ZeichenRegister pair DE points to the delimiter, register pair HL holds the 16 bit number if carry flag is not set. Carry flag is set on overflow or illegal character
DEC255.MAC
DEC255 B: Wandeln einer Zeichenkette in vorzeichenlose 8 Bit GanzzahlConvert an ASCII decimal string to unsigned 8 bit integer
  I: Registerpaar DE hält Adresse der Zeichenkette. Register B hält Endzeichen der Zeichenkette, bei -1 endet Wandlung bei erstem nichtnumerischen ZeichenRegister pair DE points to the ASCII buffer. Register B holds the end marker of the string; if -1, conversion ends at the first non-numeric character
  O: Registerpaar DE zeigt auf Abbruchzeichen, Registerpaar HL hält 16 Bit Zahl mit höherwertigem Byte Null und Akku hält 8 Bit Zahl bei nicht gesetzter Carryflag. Carryflag ist gesetzt bei Überlauf oder falschem ZeichenRegister pair DE points to the delimiter, register pair HL holds the 16 bit number with high byte zero and accumulator holds the 8 bit number if carry flag is not set. Carry flag is set on overflow (number > 255) or illegal character
SGNIN.MAC
SGNIN B: Wandeln einer Zeichenkette in 16 Bit Ganzzahl mit VorzeichenConvert a signed ASCII decimal string to 16 bit integer
  I: Registerpaar DE hält Adresse der Zeichenkette. Register B hält Endzeichen der Zeichenkette, bei -1 endet Wandlung bei erstem nichtnumerischen ZeichenRegister pair DE points to the ASCII buffer. Register B holds the end marker of the string; if -1, conversion ends at the first non-numeric character
  O: Registerpaar DE zeigt auf Abbruchzeichen, Registerpaar HL hält 16 Bit Zahl bei nicht gesetzter Carryflag. Carryflag ist gesetzt bei Überlauf oder falschem ZeichenRegister pair DE points to the delimiter, register pair HL holds the 16 bit number if carry flag is not set. Carry flag is set on overflow or illegal character

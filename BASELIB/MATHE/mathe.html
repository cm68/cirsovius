Routinen zum 16 Bit-BasisrechnenRoutines for 16 Bit Basic Arithmetic

Die hier vorgestellten Routinen sind solche für das Basisrechnen mit den 16-Bit CPU Registern. Bei einigen Routinen (z.B. SUBHD1 und SUBHD2) wird auf eine Levelabhängigkeit verwiesen. Diese ist hier erkärt.The routines presented here are those for basic arithmetic using the 16-bit CPU registers. For some routines (e.g. SUBHD1 and SUBHD2) a level dependency is referenced. This is explained here.

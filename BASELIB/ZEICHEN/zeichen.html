GETLIN.MAC
GETLIN B: Einlesen einer neuen Zeile vom KeyboardRead a new line from keyboard
  I: Das Registerpaar DE halt die Adresse auf einen Zeilenpuffer. Das erste Byte in diesem Puffer gibt die maximale Anzahl einzulesender Zeichen anReg pair DE points to a line buffer. The first byte in this buffer holds the maximum number of characters to read
  O: Das zweite Byte des Zeichenpuffers (Puffer+1) gibt die tatsächliche Anzahl gelesener Zeichen an. Die wirklichen Zeichen beginnen bei Puffer+2. Das Ende des Puffers wird durch eine Null angezeigt. Die Carry Flag ist gesetzt bei leerem PufferThe second byte of the buffer (buffer+1) holds the actual number of characters read. The actual characters start at buffer+2. The end of the buffer is indicated by a zero byte. Carry flag is set if buffer is empty

Der Puffer kann bereits belegt sein mit Text.Reg pair DE holds the address of a line buffer. The first byte in this buffer holds the maximum number of characters to read. The buffer may already contain text.
  O: Das zweite Byte des Zeichenpuffers (Puffer+1) gibt die tatsächliche Anzahl gelesener Zeichen an. Die wirklichen Zeichen beginnen bei Puffer+2. Das Ende des Puffers wird durch eine Null angezeigt. Die Carry Flag ist gesetzt bei leerem Puffer. Falls bereits Text im Puffer stand, so lässt sich dieser mit den Standard Sequenzen editierenThe second byte of the buffer (buffer+1) holds the actual number of characters read. The actual characters start at buffer+2. The end of the buffer is indicated by a zero byte. Carry flag is set if buffer is empty. If text was already in the buffer, it can be edited using the standard sequences
EMULIN.MAC
EMULIN @ETAB
@CTAB		 B: Eingabe einer Zeile vom KeyboardEmulate line input from keyboard (similar to BDOS function 10)
  I: Das Registerpaar DE hält die Adresse auf einen Zeilenpuffer. Das erste Byte in diesem Puffer gibt die maximale Anzahl einzulesender Zeichen an. Das Registerpaar HL zeigt auf eine Liste von Kontrollzeichen, die die Eingabe unterbrechen können, falls noch kein anderes Zeichen eingegeben wurde. Das erste Byte dieser Liste gibt die Anzahl Zeichen an oder ist Null, falls keine Unterbrechung erfolgen soll.
Das Register B beinhaltet Kontrollwort:
Bit 0 : Wandlung in Großbuchstaben wenn gesetzt
Bit 1 : Cursor unsichtbar machen wenn gesetzt
Bit 2 : Invers Video einschalten wenn gesetzt
@CTAB hält Kontrollzeichen für die Eingabe
@ETAB hält Cursor KontrollcodesReg pair DE holds the address of a line buffer. The first byte in this buffer holds the maximum number of characters to read. Reg pair HL points to a list of control characters that can break input if no other character has been entered yet. The first byte of this list holds the number of characters, or is zero if no break should occur.
Register B holds the control word:
Bit 0 : Convert to upper case if set
Bit 1 : Make cursor invisible if set
Bit 2 : Turn on reverse video if set
@CTAB holds control characters for input
@ETAB holds cursor control codes
  O: Das zweite Byte des Zeichenpuffers (Puffer+1) gibt die tatsächliche Anzahl gelesener Zeichen an. Die wirklichen Zeichen beginnen bei Puffer+2. Das Ende des Puffers wird durch eine Null angezeigt. Die Carry Flag ist gesetzt bei leerem Puffer, hierbei hält der Akku eine Null oder das UnterbrechungszeichenThe second byte of the buffer (buffer+1) holds the actual number of characters read. The actual characters start at buffer+2. The end of the buffer is indicated by a zero byte. Carry flag is set if buffer is empty; in that case Accu holds zero or the break character

CONSET.MAC
CONSET B: Neudefinition des KonsolmodusDefine new console mode
  I: Registerpaar DE hält 16 Bit Kontrollwort:
0 : Konsolstatus wird nur für Ctrl-C überprüft
1 : Ctrl-S und Ctrl-Q werden ignoriert
2 : Ctrl-I wird nicht expandiert, kein Druckerecho
3 : Ctrl-C führt nicht zu einem Warmstart
8 und 9 sind nur für den internen BDOS Gebrauch verfügbar, der Rest wird nicht benutzt.Reg pair DE holds a 16-bit control word:
0 : Console status is only checked for Ctrl-C
1 : Ctrl-S and Ctrl-Q are ignored
2 : Ctrl-I is not expanded, no printer echo
3 : Ctrl-C does not cause a warm start
8 and 9 are reserved for internal BDOS use, the rest is unused.
  O: Registerpaar DE hält das alte Kontrollwort, falls das Paar beim Aufruf den Wert FFFFH hatteReg pair DE holds the old control word if the pair held the value FFFFH on entry

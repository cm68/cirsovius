Routinen zum Arbeiten mit RegisternRoutines for working with registers

Die hier vorgestellten Routinen sind solche für das Arbeiten mit den 16-Bit CPU Registern.The routines presented here are for working with the 16-bit CPU registers.
Modul-EntryModule Entry B: BeschreibungDescription, I: Eingabe-Input parameter, O: AusgabeparameterOutput parameter
NEGHL.MAC
NEGHL B: Negation einer 16 Bit Ganzzahl im Register HLNegate a 16-bit integer in register HL
  I: Registerpaar HL hält 16 Bit ZahlRegister pair HL holds 16-bit number
  O: Registerpaar HL hält das Zweierkomplement der Zahl (-HL)Register pair HL holds the two's complement of the number (-HL)
NEGDE.MAC
NEGDE B: Negation einer 16 Bit Ganzzahl im Register DENegate a 16-bit integer in register DE
  I: Registerpaar DE hält 16 Bit ZahlRegister pair DE holds 16-bit number
  O: Registerpaar DE hält das Zweierkomplement der Zahl (-DE)Register pair DE holds the two's complement of the number (-DE)
SHFRHL.MAC
SHFRHL B: Schieben eines 16 Bit Wertes eine Stelle nach rechtsShift a 16-bit value one position to the right (divide by 2)
@SHRHL I: Registerpaar HL hält 16 Bit WertRegister pair HL holds 16-bit value
  O: Registerpaar HL hält den 16 Bit Wert dividiert durch zwei. Beim Aufruf SHFRHL wird die Carry Flag vorher auf Null gesetzt, beim Aufruf @SHRHL wird diese mit benutzt.Register pair HL holds the 16-bit value divided by two. When calling SHFRHL, the carry flag is cleared first; when calling @SHRHL, the carry flag is included in the shift.

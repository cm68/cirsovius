Routinen zum Arbeiten mit 32-Bit ZahlenRoutines for working with 32-bit numbers

Die hier vorgestellten Routinen sind solche für die Verabeitung von Routinen mit 32-Bit Zahlen. Einige Hinweise zum Format findet sich hier.The routines presented here are those for processing 32-bit numbers. Some notes on the format can be found here.
Modul-EntryModule Entry B: BeschreibungDescription, I: Eingabe-Input, O: AusgabeparameterOutput parameters
ADD32.MAC
ADD32 B: Addition zweier ZahlenAdd two 32-bit integers
  I: Registerpaar BC zeigt auf erste Zahl, Paar DE auf zweite und Paar HL auf ErgebnisRegister pair BC points to first number, pair DE points to second number, and pair HL points to result
  O: Summe in Ergebnis abgelegt. Bei Überlauf ist die Carry Flag gesetzt.Sum stored in result. Carry flag is set on overflow.
SUB32.MAC
SUB32 B: Subtraktion zweier ZahlenSubtract two 32-bit integers
  I: Registerpaar BC zeigt auf erste Zahl, Paar DE auf Subtrahenden und Paar HL auf ErgebnisRegister pair BC points to first number, pair DE points to subtrahend, and pair HL points to result
  O: Differenz in Ergebnis abgelegt. Bei Überlauf ist die Carry Flag gesetzt.Difference stored in result. Carry flag is set on overflow.
MUL32.MAC
MUL32 B: Multiplikation zweier ZahlenMultiply two 32-bit integers
  I: Registerpaar BC zeigt auf erste Zahl, Paar DE auf zweite und Paar HL auf ErgebnisRegister pair BC points to first number, pair DE points to second number, and pair HL points to result
  O: Produkt in Ergebnis abgelegt. Bei Überlauf ist die Carry Flag gesetzt.Product stored in result. Carry flag is set on overflow.
DIV32.MAC
DIV32 B: Division zweier ZahlenDivide two 32-bit integers
  I: Registerpaar BC zeigt auf Dividenden, Paar DE auf Divisor und Paar HL auf ErgebnisRegister pair BC points to dividend, pair DE points to divisor, and pair HL points to result
  O: Quotient in Ergebnis abgelegt. Bei Überlauf oder Division durch Null ist die Carry Flag gesetzt.Quotient stored in result. Carry flag is set on overflow or division by zero.
MOD32.MAC
MOD32 B: Ermitteln des Restes zweier 32 Bit ZahlenGet remainder of two 32-bit numbers
  I: Registerpaar BC zeigt auf Dividenden, Paar DE auf Divisor und Paar HL auf ErgebnisRegister pair BC points to dividend, pair DE points to divisor, and pair HL points to result
  O: Rest in Ergebnis abgelegt. Bei Überlauf, negativem Divisor oder Division durch Null ist die Carry Flag gesetzt.Remainder stored in result. Carry flag is set on overflow, negative divisor, or division by zero.
NEG32.MAC
NEG32 B: Negation einer ZahlNegate a number
  I: Registerpaar HL zeigt auf die ZahlRegister pair HL points to the number
  O: Zahl negiertNumber filled with negated value

SHL32 B: Links schieben einer ZahlShift left a number
  I: Registerpaar HL zeigt auf die ZahlRegister pair HL points to the number
  O: Zahl um eine Stelle nach links geschobenNumber shifted left by one position
SHR32.MAC
SHR32 B: Rechts schieben einer ZahlShift right a number
  I: Registerpaar HL zeigt auf die ZahlRegister pair HL points to the number
  O: Zahl um eine Stelle nach rechts geschobenNumber shifted right by one position
DIN32.MAC
DIN32 B: Umwandlung einer Zeichenkette in eine vorzeichenlose 32 Bit ZahlConvert an ASCII decimal string to an unsigned 32-bit number
  I: Registerpaar DE zeigt auf die Zeichenkette, HL zeigt auf die ganze Zahl und Register B hält das Endzeichen der Kette. Steht in Register B -1, dann wird die Wandlung beim ersten nicht-numerischen Zeichen beendet.Register pair DE points to the string, HL points to the integer, and register B holds the end marker of the string. If register B holds -1, conversion ends at the first non-numeric character found.
  O: Resultat in der ganzen Zahl abgelegt. Bei Zahlenfehler oder Überlauf ist die Carry Flag gesetzt. Das Registerpaar zeigt auf das nächste Zeichen hinter der ZahlResult stored in the integer. Carry flag is set on numeric error or overflow. Register pair DE points to the next character after the number.
SIN32.MAC
SIN32 B: Umwandlung einer Zeichenkette in eine 32 Bit Zahl mit VorzeichenConvert an ASCII decimal string to a signed 32-bit number
  I: Registerpaar DE zeigt auf die Zeichenkette, HL zeigt auf die ganze Zahl und Register B hält das Endzeichen der Kette. Steht in Register B -1, dann wird die Wandlung beim ersten nicht-numerischen Zeichen beendet.Register pair DE points to the string, HL points to the integer, and register B holds the end marker of the string. If register B holds -1, conversion ends at the first non-numeric character found.
  O: Resultat in der ganzen Zahl abgelegt. Bei Zahlenfehler oder Überlauf ist die Carry Flag gesetzt. Das Registerpaar zeigt auf das nächste Zeichen hinter der ZahlResult stored in the integer. Carry flag is set on numeric error or overflow. Register pair DE points to the next character after the number.
DOUT32.MAC
DOUT32 B: Wandlung einer vorzeichenlosen 32 Bit Zahl in eine ZeichenketteConvert an unsigned 32-bit number to an ASCII decimal string
  I: Registerpaar DE zeigt auf die Zeichenkette, HL zeigt auf die ganze Zahl und Register B hält das Endzeichen der KetteRegister pair DE points to the string, HL points to the integer, and register B holds the end marker of the string
  O: Zeichenkette abgeschlossen mit Zeichen in Register B. Das Registerpaar HL zeigt auf den AbschlussString terminated with the character in register B. Register pair HL points to the terminator.
SOUT32.MAC
SOUT32 B: Wandlung einer 32 Bit Zahl mit Vorzeichen in eine ZeichenketteConvert a signed 32-bit number to an ASCII decimal string
  I: Registerpaar DE zeigt auf die Zeichenkette, HL zeigt auf die ganze Zahl und Register B hält das Endzeichen der KetteRegister pair DE points to the string, HL points to the integer, and register B holds the end marker of the string
  O: Zeichenkette abgeschlossen mit Zeichen in Register B. Das Registerpaar HL zeigt auf den AbschlussString terminated with the character in register B. Register pair HL points to the terminator.
C1632.MAC
C1632 B: Expandieren einer 16 Bit Zahl in eine 32 Bit Zahl mit VorzeichenExpand a signed 16-bit number to a 32-bit number
  I: Registerpaar HL hält die 16 Bit Zahl, Registerpaar DE zeigt auf die 32 Bit ZahlRegister pair HL holds the 16-bit number, register pair DE points to the 32-bit number
  O: 32 Bit Zahl aus 16 Bit Zahl konstruiert32-bit number constructed from 16-bit number
C3216.MAC
C3216 B: Verkürzen einer 32 Bit Zahl in eine 16 Bit ZahlTruncate a 32-bit number to a 16-bit number
  I: Registerpaar DE zeigt auf die 32 Bit ZahlRegister pair DE points to the 32-bit number
  O: Registerpaar HL hält die 16 Bit ZahlRegister pair HL holds the 16-bit number
C32FL.MAC
C32FL B: Wandeln einer 32 Bit Zahl in eine FließkommazahlConvert a 32-bit number to a floating point number
  I: Registerpaar DE zeigt auf die 32 Bit Zahl, HL zeigt auf die FließkommazahlRegister pair DE points to the 32-bit number, HL points to the floating point number
  O: Fließkommazahl konstruiert aus ganzer ZahlFloating point number constructed from integer
CFL32.MAC
CFL32 B: Wandeln einer Fließkommazahl in eine 32 Bit ZahlConvert a floating point number to a 32-bit number
  I: Registerpaar HL zeigt auf die 32 Bit Zahl, DE zeigt auf die FließkommazahlRegister pair HL points to the 32-bit number, DE points to the floating point number
  O: Ganze Zahl konstruiert aus Fließkommazahl. Carry Flag gesetzt bei nicht möglicher KonvertierungInteger constructed from floating point number. Carry flag set if conversion is not possible.

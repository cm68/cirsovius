Routinen für DatenbackupRoutines for Data Backup

Die hier vorgestellten Routinen sind solche für Datenbackup:The routines presented here are for data backup:
Modul-EntryModule Entry B: BeschreibungDescription, I: Eingabe-Input parameters, O: AusgabeparameterOutput parameters
CREBAK.MAC
CREBAK $$BAK B: Erstellen einer neuen Datei mit Extension .BAKCreate a new file with extension .BAK
  I: Registerpaar DE hält Adresse der neuen FCB, Registerpaar HL hält Adresse der alten FCBRegister pair DE holds address of the new FCB, register pair HL holds address of the original FCB
  O: Die Carryflag ist gesetzt im Fehlerfall, falls die alte Datei auch die Extension .BAK auf gleichem Laufwerk hat oder falls eine neue Datei nicht eingerichtet werden kann. Bei nicht gesetzter Carryflag ist eine neue Datei eingerichtet. Der Akku hält im Fehlerfall einen KodeCarry flag is set on error if the original file also has extension .BAK on the same drive, or if a new file cannot be created. Only the name field is copied to the new FCB, so different drives may be used. Any existing file with extension .BAK will be deleted. When carry flag is clear, a new file has been created. On error, the accumulator holds a code: 0 if the original file does not exist, 1 if .BAK cannot be created, 2 on extension .BAK and same drive
DELBAK.MAC
DELBAK B: Löschen einer Datei und Umbenennen einer anderen Datei in Namen der gelöschten DateiDelete a file and rename another file to the name of the deleted file
  I: Registerpaar DE hält Adresse der FCB1, die umbenannt werden soll. Registerpaar HL hält Adresse der FCB2, die gelöscht werden soll.Register pair DE holds address of FCB1, which is to be renamed. Register pair HL holds address of FCB2, which is to be deleted.
  O: Datei mit FCB2 gelöscht, FCB1 umbenannt in Namen von FCB2. Carryflag gesetzt, falls Umbenennung fehlerhaft, sonst nicht gesetzt.File with FCB2 deleted, FCB1 renamed to the name of FCB2. Carry flag set if rename failed, otherwise clear.
SWPNAM.MAC
SWPNAM B: Umbenennen zweier Dateien in den Namen der jeweils anderen DateiSwap the names of two files
  I: Registerpaar DE hält FCB1 und Registerpaar HL hält FCB2Register pair DE holds FCB1 and register pair HL holds FCB2
  O: Erste Datei umbenannt in Namen der zweiten Datei und umgekehrt. Carryflag gesetzt falls Fehler bei einer Umbenennung, d.h. eine Datei ist nicht auf DisketteFirst file renamed to name of the second file and vice versa. Carry flag set if error during a rename, i.e. a file is not on disk
MOVNAM.MAC
@MVNAM B: Kopieren eines Namensfeldes innerhalb einer FCBCopy a name field within an FCB
  I: Registerpaar DE hält Adresse der FCB und Registerpaar HL hält Adresse auf ein 16 Byte NamensfeldRegister pair DE holds address of the FCB and register pair HL holds address of a 16-byte name field
  O: Inhalt ab FCB+16 belegt mit NamensfeldContents starting at FCB+16 filled with the name field

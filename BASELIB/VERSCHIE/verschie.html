Hilfs-RoutinenUtility Routines

Die hier vorgestellten Routinen sind verschiedene Hilfs-Routinen. Bei einigen Routinen (z.B. MOVE1, MOVE2 und MOVE3) wird auf eine Levelabhängigkeit verwiesen. Diese ist hier erkärt.The routines presented here are various utility routines. For some routines (e.g. MOVE1, MOVE2 and MOVE3), a level dependency is referenced. This is explained here.

Byte 1 :Zeichen 1
 ..
Byte n :Zeichen n
Wort 1 :Adresse der Routine für Zeichen n
 ..
Wort n :Adresse der Routine für Zeichen 1

Byte 0 :Number of characters in the table
Byte 1 :Character 1
 ..
Byte n :Character n
Word 1 :Address of the routine for character n
 ..
Word n :Address of the routine for character 1

  O: Sprung auf die entsprechende Routine oder weiter mit Instruktion hinter Tabellenzeiger, falls keine Zeichenübereinstimmung vorliegtJump to the corresponding routine, or continue with the instruction after the table pointer if no character match is found

CMPITM.MAC
CMPITM B: Vergleich einer Zeichenkette mit einem Datenfeld variabler LängeCompare a character string with a variable-length data field
  I: Registerpaar HL zeigt auf das Datenfeld, dessen Elemente jeweils mit der Länge beginnen und dessen letzte Eintragung eine Null sein muss. Registerpaar DE zeigt auf die zu suchende Zeichenkette, dessen Länge im Register C enthalten sein muss.Register pair HL points to the data field, whose elements each start with the length and whose last entry must be zero. Register pair DE points to the character string to search for, whose length must be held in register C.
  O: Zeroflag nicht gesetzt falls Vergleich nicht übereinstimmt. Bei gleichen Werten ist Zeroflag gesetztZero flag is reset if comparison does not match. Zero flag is set if values are equal

MEMMOV.MAC
MEMMOV B: Speicher kopieren mit Test, ob genügend Speicher vorhanden ist.Copy memory with check for sufficient available space.
  I: Register HL zeigt auf Quelle, Register DE zeigt auf Senke und Register BC hält Anzahl zu kopierender Bytes.
Vor Aufruf sind folgende Inhalte festzulegen:
MEMMOV-2: Adresse einer Fehlermeldung für Konsolausgabe (Voreinstellung 0000H, keine Fehlermeldung)
MEMMOV-4: Adresse, die auf höchste Speicheradresse zeigt (Voreinstellung 0006H, TPA Obergrenze)
MEMMOV-6: Adresse einer Routine, auf die nach einem Fehler verzweigt wird (Voreinstellung 0000H, Warmstart)
Register HL points to source, register DE points to destination, and register BC holds the number of bytes to copy.
Before calling, the following contents must be set:
MEMMOV-2: Address of an error message for console output (default 0000H, no error message)
MEMMOV-4: Address pointing to the highest memory address (default 0006H, TPA upper limit)
MEMMOV-6: Address of a routine to branch to after an error (default 0000H, warm start)
  O: ---

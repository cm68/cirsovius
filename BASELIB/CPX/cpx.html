Lade-RoutinenLoader Routines

Die hier vorgestellten Routinen sind solche für die Verabeitung von speziellen Routinen für Overlays und .PRL-Dateien.The routines presented here are those for processing special routines for overlays and .PRL files.
Modul-EntryModule Entry B: Beschreibung, I: Eingabe-, O: AusgabeparameterDescription, I: Input, O: Output parameters
FCBSUP.MAC
FCBSUP B: Wandeln einer festen Zeichenkette in eine FCBGet file FCB from a fixed length file name
  I: Das Register HL weist auf eine Zeichenkette im festen Format, Register DE weist auf eine FCB. Das feste Format hat folgenden Aufbau: acht Zeichen für den Dateinamen, einem Punkt und drei Zeichen für den Dateityp, gefolgt von einer NullRegister pair HL points to a fixed length file name, register pair DE points to an FCB. The fixed format is defined as eight characters for the file name, one dot and three characters for the extension, all closed by zero (resulting in a total of 13 characters)
  O: Die Carryflag ist gesetzt bei einem Fehler in der ZeichenketteCarry flag is set on error in the fixed format string
SELITM.MAC
SELITM B: Auswahl mittels BildschirmkontrollenSelect item from list using screen controls
  I: Das Register HL weist auf eine Liste aus Zeichenketten mit gleichem Format, Register DE weist auf einen Parameterblock und Register BC hält die Anzahl in der Liste. Die Elemente müssen mit Null enden. Der Parameterblock besteht aus 11 BytesRegister pair HL points to a list of strings of equal format, register pair DE points to a parameter block and register pair BC holds the number of items in the list. Each item must be closed by zero. The parameter block consists of 11 bytes
  O: Die Carryflag ist gesetzt, wenn das Zeichen für Abbruch erkannt wurde. Bei RETURN hält das Register DE die Adresse des angewählten Elementes und das Register HL die Koordinaten des Cursors unmittelbar hinter der letzten Ausgabezeile.Carry flag is set if the break character was detected. On RETURN, register pair DE holds the address of the selected item and register pair HL holds the cursor coordinates immediately after the last output line.
LOADER.MAC
LODREC B: Aktivieren der RSX LOADERRecover the CP/M PLUS LOADER RSX for using the extended loader functions calling through BDOS
  I: Das Register DE hält Null oder eine Adresse auf eine FehlermeldungRegister pair DE holds zero or an address pointing to an error message
  O: Wenn die RSX nicht gefunden wurde, so wird bei Null im Register DE die Carryflag gesetzt. Andernfalls erfolgt die Fehlermeldung und Beendigung des Programmes. Bei nicht gesetzter Carryflag stehen Erweiterungen des Betriebssystems zur VerfügungIf the RSX was not found: when register DE is zero, the carry flag is set. Otherwise the error message is printed and the program terminates. If carry flag is reset, the extended operating system functions are available
OVLMNGR.MAC
?OVLA0 B: Overlay Routine für Overlays, die mit LINK generiert wurdenOverlay manager for programs linked with DIGITAL RESEARCH's LINK program
(?OVLAY) I: Das Register BC weist auf den Dateinamen des zu ladenden Overlays. Register DE und HL halten optionale Parameter für das aufzurufende Overlay.Register pair BC holds the filename of the overlay to be loaded (8 characters). Registers DE and HL hold optional parameters to pass to the overlay.
  O: Wird durch das aufgerufene Overlay bestimmtAs defined by the called overlay
PRLPRP.MAC
PRLPRP B: Ladevorbereitung für eine Page ReLocatable (PRL) DateiPrepare the load of a Page ReLocatable (PRL) file
  I: Das Registerpaar HL weist auf die erste mögliche Ladeadresse der PRL Datei, DE zeigt auf den File Control Block (FCB) der zu ladenden Datei.Register pair HL points to the first possible load address of the PRL file, register pair DE points to the File Control Block (FCB) of the file to be loaded.
  O: Das Registerpaar HL weist auf die justierte Ladeadresse (Pagebeginn, Page eventuell inkrementiert), DE weist auf die Adresse nach dem zu ladenden Programm. Die Carryflag ist gesetzt, wenn die Datei nicht gefunden wurde oder bei zu großer Datei.Register pair HL points to the adjusted load address (page boundary, page possibly incremented), register pair DE points to the address after the program to be loaded. Carry flag is set if the file was not found or on memory overflow.
LODPRL.MAC
LODPRL B: Laden einer Page ReLocatable (PRL) DateiLoad a Page ReLocatable (PRL) file
  I: Das Registerpaar HL weist auf die Ladeadresse der PRL Datei, DE zeigt auf den File Control Block (FCB) der zu ladenden Datei.Register pair HL points to the load address of the PRL file, register pair DE points to the File Control Block (FCB) of the file to be loaded.
  O: Im Fehlerfall ist das Zero Flag nicht gesetzt. Dann liegt entweder ein Ladefehler vor oder die benötigte System RSX LOADER ist nicht installiert. Im Erfolgsfall ist die Zero Flag gesetzt und Register HL weist auf die Ladadresse der PRL.On error the Zero flag is not set. This indicates either a load error or that the required system RSX LOADER is not installed. On success the Zero flag is set and register pair HL points to the load address of the PRL.
INIMOD.MAC
INIMOD B: Änderung geräteabhängiger KontrollzeichenChange definition of device dependent control characters
  I: Der Akku hält das Zeichen ‚i' oder ‚I' und das Registerpaar HL zeigt auf einen ParameterblockThe accumulator holds the character 'i' or 'I' and register pair HL points to a parameter block
  O: Falls der Akku nicht das angegebene Zeichen hält, wird keine Änderung durchgeführtIf the accumulator does not hold the specified character, no change is performed

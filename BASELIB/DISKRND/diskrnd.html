Routinen zum Arbeiten mit RekordsRoutines for working with records

Die hier vorgestellten Routinen sind solche für zur Rekord Ein-/Ausgabe mit variabler Länge bei Diskettenoperationen. Einige Hinweise zu den Routinen finden sich hier.The routines presented here are for record input/output with variable length during disk operations. Some notes on the routines can be found here.
Modul-EntryModule Entry B: BeschreibungDescription, I: Eingabe-Input, O: AusgabeparameterOutput parameters
CRERND.MAC
CRERND B: Erstellen einer Leerdatei mit Rekords variabler LängeCreate an empty file with variable length records
  I: Akku hält Länge des Rekords im Bereich 1..255
Registerpaar BC hält Rekordnummer im Bereich 1..65535
Registerpaar DE hält Adresse der FCB
Registerpaar HL hält Adresse eines DMA Puffers für die Einstellung eines Rekords Accu holds record length in the range 1..255
Reg pair BC holds record number in the range 1..65535
Reg pair DE holds address of the FCB
Reg pair HL holds address of a DMA buffer for setting up a record
  O: Die Carryflag ist gesetzt falls Datei schon vorhanden ist oder bei Disketten- oder Rekordparameterfehler, sonst nicht gesetzt. Die Datei ist zum Lesen geöffnetCarry flag is set if file already exists or on disk or record parameter error, otherwise not set. File is opened for reading

Reg pair BC holds record number in the range 1..65535

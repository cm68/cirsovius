Zeit-RoutinenDate/Time Routines

Die hier vorgestellten Routinen sind solche für die Verabeitung des Datums. Sie sind nicht auf das Jahr-2000-Problem überprüft:The routines presented here are used for processing dates and times. They have not been verified for the Year-2000 problem:
Modul-EntryModule Entry B: BeschreibungDescription, I: Eingabe-Input, O: AusgabeparameterOutput parameters
MMDDYY.MAC
MMDDYY B: Aktuelles Datum in Zeichenkette wandeln, Format MM/TT/JJConvert current date to string, format MM/DD/YY
  I: Register HL zeigt auf Zeichenkette, Register B hält das Endzeichen (typischerweise „$" oder Null) und Register C hält das Trennungszeichen zwischen den einzelnen Darstellungen.Register HL points to string, register B holds the end character (typically "$" or null), and register C holds the delimiter between the individual fields.
  O: ---Register HL points to end character
DDMMYY.MAC
DDMMYY B: Aktuelles Datum in Zeichenkette wandeln, Format TT MMM JJConvert current date to string, format DD MMM YY
  I: Register HL zeigt auf Zeichenkette, Register B hält das Endzeichen (typischerweise „$" oder Null) und Register C hält das Trennungszeichen zwischen den einzelnen Darstellungen. In der Adresse DDMMYY-2 steht ein Zeiger auf eine Monatstabelle (je drei Zeichen), Voreinstellung ist englisch.Register HL points to string, register B holds the end character (typically "$" or null), and register C holds the delimiter between the individual fields. At address DDMMYY-2 there is a pointer to a month name table (three characters each), default is English.
  O: ---Register HL points to end character

PUTBCD B: BCD Zeichen in Zeichenkette wandelnStore BCD byte into two ASCII characters in a string

Routinen zur Verabeitung von ZeichenRoutines for character processing

Die hier vorgestellten Routinen sind solche für die Verabeitung von Zeichen:The routines presented here are for the processing of characters:
Modul-EntryModule Entry B: Beschreibung, I: Eingabe-, O: AusgabeparameterDescription, I: Input, O: Output parameters
SKPBLK.MAC
SKPBLK B: Überspringen von LeerzeichenSkip blanks in ASCII stream
  I: Registerpaar DE hält Adresse des ZeichenpuffersRegister pair DE points to the string buffer
  O: Registerpaar DE hält Adresse auf erstes Zeichen, das kein Leerzeichen ist (Leerzeichen oder Tabulator)Register pair DE points to the first non-blank character (not a space or tab)
SKPITM.MAC
SKPITM B: Überspringen von Zeichen außer LeerzeichenSkip non-blank characters in ASCII stream
  I: Registerpaar DE hält Adresse des ZeichenpuffersRegister pair DE points to the string buffer
  O: Registerpaar DE hält Adresse auf erstes Zeichen, das ein Leerzeichen istRegister pair DE points to the first blank character
TSTLET.MAC
TSTLET B: Test ob Zeichen im Bereich A .. Z liegtTest if character is in range A .. Z
  I: Akku hält das ZeichenAccumulator holds the character
  O: Carryflag ist gesetzt falls das Zeichen nicht im Bereich liegt, sonst nicht gesetztCarry flag is set if the character is not in range, otherwise reset
TSTDIG.MAC
TSTDIG B: Test ob Zeichen im Bereich 0 .. 9 liegtTest if character is in range 0 .. 9 and get binary value
  I: Akku hält das ZeichenAccumulator holds the character
  O: Carryflag ist gesetzt falls das Zeichen nicht im Bereich liegt, sonst nicht gesetzt. Akku hält das Binäräquivalent der Ziffer bei nicht gesetzter Carryflag, sonst keine ÄnderungCarry flag is set if the character is not in range, otherwise reset. Accumulator holds the binary equivalent of the digit when carry is reset, otherwise unchanged
UPPCON.MAC
UPPCON B: Wandeln eines Zeichens im Bereich a .. z in Großbuchstaben A .. ZConvert a character in range a .. z to upper case A .. Z
  I: Akku hält das ZeichenAccumulator holds the character
  O: Akku hält den Großbuchstaben bei nicht gesetzter Carryflag. Bei gesetzter Carryflag keine Änderung im AkkuAccumulator holds the upper case character when carry is reset. When carry is set, accumulator is unchanged
UPPLIN.MAC
UPPLIN B: Wandeln von Zeichen im Bereich a .. z in Großbuchstaben A .. Z aus einer ZeichenketteConvert characters in range a .. z to upper case A .. Z in a string
  I: Registerpaar DE hält Adresse der Zeichenkette. Register B hält das Endzeichen in der KetteRegister pair DE points to the string. Register B holds the end-of-string character
  O: Registerpaar DE hält Adresse der gewandelten ZeichenketteString converted to upper case
LOWCON.MAC
LOWCON B: Wandeln eines Zeichens im Bereich A .. Z in Kleinbuchstaben a .. zConvert a character in range A .. Z to lower case a .. z
  I: Akku hält das ZeichenAccumulator holds the character
  O: Akku hält den Kleinbuchstaben bei nicht gesetzter Carryflag. Bei gesetzter Carryflag keine Änderung im AkkuAccumulator holds the lower case character when carry is reset. When carry is set, accumulator is unchanged
LOWLIN.MAC
LOWLIN B: Wandeln von Zeichen im Bereich A .. Z in Kleinbuchstaben a .. z aus einer ZeichenketteConvert characters in range A .. Z to lower case a .. z in a string
  I: Registerpaar DE hält Adresse der Zeichenkette. Register B hält das Endzeichen in der KetteRegister pair DE points to the string. Register B holds the end-of-string character
  O: Registerpaar DE hält Adresse der gewandelten ZeichenketteString converted to lower case
CMDARG.MAC
CMDARG B: Zerlegen einer Kommandozeile in ArgumenteParse a command line into arguments
  I: Registerpaar HL zeigt auf ein zu füllendes Adressfeld. Registerpaar DE zeigt auf Kommandozeile, das erste Byte gibt die Anzahl Zeichen in der Zeile an. Register B hält die maximal erlaubten Argumente.Register pair HL points to a string pointer array to be filled. Register pair DE points to the command line, where the first byte gives the number of characters in the line. Register B holds the maximum allowed arguments.
  O: Das Addressfeld ist gefüllt, die einzelnen Argumente in der Kommandozeile sind abgeschlossen mit Nullen. Der Akku hält die Anzahl gefundener Argumente. Die Carryflag zeigt an, dass kein Argument oder zu viele Argumente gefunden worden.The address array is filled, the individual arguments in the command line are terminated with zeros. The accumulator holds the number of arguments found. The carry flag indicates that no arguments or too many arguments were found.
INCASC.MAC
INCASC B: Inkrementieren einer numerischen Zeichenkette.Increment an ASCII coded numeric string.
  I: Registerpaar HL zeigt auf das niederwertigste Zeichen. Register B hält die Anzahl Zeichen in der Kette.Register pair HL points to the least significant digit. Register B holds the number of characters in the string.
  O: Zeichenkette um 1 erhöht. Alle Zeichen außerhalb des Bereichs 0..9 werden mit 1 initiiert.String incremented by 1. All characters outside the range 0..9 are initialized to 1.
SUMASC.MAC
SUMASC B: Addieren zweier numerischen Zeichenketten.Add two ASCII coded numeric strings.
  I: Registerpaar HL zeigt auf das niederwertigste Zeichen der ersten und Registerpaar DE auf das der zweiten Zeichenkette. Register B hält die Anzahl Zeichen in der Kette.Register pair HL points to the least significant digit of the first string and register pair DE to that of the second string. Register B holds the number of characters in the string.
  O: Summe ist in der zweiten Zeichenkette abgelegt.Sum is stored in the second string.
